# Todo App

A simple and calm task manager built with React and Vite. It lets you add, edit, complete, and delete tasks, with a soft glass-style interface and local persistence so your tasks stay available after refresh.

## Features

- Add new tasks with title, description, priority, and dates
- Edit existing tasks
- Mark tasks as done
- Delete tasks
- View tasks by priority

## How to run

1. Install dependencies: npm install
2. Start the app: npm run dev
3. Open the local URL shown in the terminal
